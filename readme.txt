Coupon reveal for Gutenberg blocks page builder.

== Description ==

ZIOR Coupon Reveal is a WordPress plugin that gives website owner a way to monitize their website with affiliate marketing by way of coupons.

Features

* Unlimited coupons
* Custom style for coupon reveal button
